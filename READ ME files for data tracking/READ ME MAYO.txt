MAYO Data Folder - READ ME
The entire dataset in this folder (3mmB30) was downloaded from the public Box found at this link: https://aapm.app.box.com/s/eaw4jddb53keg1bptavvvd1sf4x3pe9h.

It can be seen that in this folder there are: 10 folders starting with the prefix 'L', each holding scans of a specific patient. The data was downloaded from the corresponding folder (3mmB30, quarter is 3mm and B30 is the kernel) within the parent folder in the public Box named: Patient_Data/Training_Image_Data.

The percentage of each folder out of the entire dataset:
L067 - 9%
L096 - 13%
L109 - 5%
L143 - 9%
L192 - 10%
L286 - 8%
L291 - 14%
L310 - 8%
L333 - 10%
L506 - 8%

As we documented regarding the split ratios in the report, for our project we chose L506 as the test set (8%), and L109 for the validation set (5%), with all the rest assigned to the training set.
